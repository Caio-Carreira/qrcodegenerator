# QR Code Lince Farm

## 📌 Descrição

O **QR Code Lince Farm** é uma aplicação gráfica em Python que permite gerar QR Codes personalizados com facilidade. Você pode definir o texto ou URL, a cor do QR Code e adicionar um logo centralizado, criando um QR Code exclusivo e estilizado.

---

## 🚀 Funcionalidades

* ✅ Gerar QR Codes personalizados com texto ou URL.
* ✅ Escolher a cor do QR Code (HEX).
* ✅ Adicionar um logo opcional ao centro do QR Code.
* ✅ Pré-visualização em tempo real do QR Code.
* ✅ Salvar o QR Code em alta resolução (1080x1080).

## 📦 Como Instalar e Usar

1. Vá até a pasta `dist/`.
2. Dê dois cliques no arquivo `qrcodegenerator.exe`.
3. O programa abrirá diretamente, sem necessidade de instalação.

---

## 🚀 Como Rodar o Projeto via Código Fonte

1. Instale o Python 3.10 ou superior.
2. No terminal, instale as dependências:

   ```bash
   pip install -r src/requirements.txt
   ```
3. Execute o aplicativo:

   ```bash
   python src/qrcodegenerator.py
   ```
